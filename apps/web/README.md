## Getting Started

First, run the development server:

```bash
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

## Install Chromium

```bash
npx @puppeteer/browsers --path ~/.cache/puppeteer/ install chromium@latest
```
